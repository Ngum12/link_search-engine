Link Library Guide

Everything you need to know to organize your links effectively

üìù Basic Usage

Link Library helps you organize all your important links in one place. The extension has three main tabs:

  • Browse: View and access all your links organized by category
  • Add New: Create new links and categories
  • Settings: Export, import, and manage your link collection
Pro Tip: Use the search bar at the top to quickly find links across all categories.

‚ûï Adding Links

To add a new link:

  1. Click on the Add New tab
  2. Enter a descriptive title (e.g., "GitHub Documentation")
  3. Paste the complete URL (including https://)
  4. Select a category from the dropdown or create a new one
  5. Add a brief description of what the link contains
  6. Add relevant tags separated by commas (e.g., "git, code, repository")
  7. Click Add Link to save
Naming Format: Use clear, descriptive titles that indicate exactly what the resource is. For example, use "MDN JavaScript Reference" rather than just "JavaScript."

Proper categorization makes finding links easier:

  • Create specific categories that make sense for your workflow
  • Default categories include: Automation, Vision, Maps, etc.
  • To add a new category, go to Add New tab and use the "Add Category" section
  • Keep category names short but descriptive (e.g., "AI Tools" instead of just "AI")

üè∑Ô∏è Using Tags Effectively

Tags improve searchability and organization:

  • Add 3-5 relevant tags to each link
  • Use specific terms related to the link's content
  • Include alternative terms people might search for
  • Example: For a Python tutorial, use tags like "python, programming, tutorial, learning"

üîç Search Techniques

Find links quickly using search:

  • Search examines titles, descriptions, URLs, and tags
  • Use the category filter dropdown to narrow results to a specific category
  • Search results are highlighted to make matches easier to spot

üíæ Backup and Transfer

Keep your links safe:

  • In Settings, use "Export Data" to create a backup file
  • Use "Import Data" to restore from a backup or transfer to another device
  • "Sync with Browser Bookmarks" lets you import your existing Chrome bookmarks
  • Regular backups are recommended to prevent data loss
